## Projects

**Dentista na Nuvem**

Full-stack platform for the NGO Turma do Bem, connecting underserved patients with volunteer dentists across Latin America.

`Java` `Quarkus` `Oracle DB` `React` `TypeScript` `Vite` `Azure App Service` `Vercel`

Layered architecture (Resource → BO → DAO → Entity), 16 REST endpoints, CORS, Oracle FIAP schema integration.

---

**AstroCare Remote Health**

Remote health system with Java/Quarkus backend, React frontend and full Oracle SQL modeling.

`Java` `Quarkus` `React` `TypeScript` `Oracle SQL` `ReportLab` `Astah UML`

UML documentation (use case with include/extend stereotypes), automated PDF generation, 10-table relational schema.
